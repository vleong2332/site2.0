The Stack

This was built with the trifecta of web technology: HTML, CSS, JS. I used gulp to help with the build process.

The Process

Since I had to get this up and running in 2 days, the wireframe and mockups were done with pencil and paper. After approval, I built the website and the client (the VP) filled out the content.
